# Navrachna School Website

A modern, responsive, and feature-rich school management website built with **React 19**, **Vite**, and **Tailwind CSS**. This project features a clean UI, smooth animations with **Framer Motion**, and a robust **Firebase** backend for administrative tasks.

## 🚀 Features

- **Dynamic Home Page**: Hero section with typing headlines and key feature highlights.
- **School Infrastructure**: Detailed pages for Classrooms, Computer Labs, and Library.
- **Administrative Pages**: Digitalized Fee Structure, Holiday Lists, and Mandatory Public Disclosures.
- **Admissions Portal**: Easy access to admission guidelines and contact information.
- **Interactive Gallery**: A beautiful masonry/grid gallery to showcase school events.
- **Admin Dashboard**: Secure login area for managing announcements, gallery images, and school details.
- **WhatsApp Integration**: Floating contact button for instant parent-school communication.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Backend/Auth**: Firebase
- **Routing**: React Router DOM (v7)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Adarsh-2415/Navrachna_school.git
   cd Navrachna_school
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run in development mode**:
   ```bash
   npm run dev
   ```

## 🔐 Admin Access

The admin panel is accessible via the `/admin` route. Authorized users can manage announcements, gallery images, and school details.
